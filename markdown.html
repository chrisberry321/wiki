Markdown is a language that helps developers create web pages quickly. Remember Markdown is a way to format text to be viewed in a browser, but we can also add in active links to image files, video files, and other web pages.

Headers are constructed with # symbols and there are six sizes with one being the largest and 6 the smallest.

  • # Best Chocolate Chip Cookies
  • ## Best Chocolate Chip Cookies
  • ### Best Chocolate Chip Cookies
  • #### Best Chocolate Chip Cookies
  • ##### Best Chocolate Chip Cookies
  • ###### Best Chocolate Chip Cookies

Any text that you would like to be in italics should be surrounded by single underscores, _. For example, _Be warned_, these will fly off of the plate!

In Markdown there are ordered lists, which means they are numbered, and there are unordered lists, which use bullet points instead of numbers. In an unordered list, we use the * symbol followed by one space before the text. In an ordered list, we use regular numerals followed by a period and a space.

  • * Butter
  • * Sugar
  • 2. Mix the flour...
  • In Markdown, we put two ** around any text that we would like bolded. For example: Bake for **10 minutes** or until golden brown.

A link is composed of two elements, the text that will display on the page surrounded by [] and the web address of where the link will go surrounded by (). For example, [Click here](http://allrecipes.com/) to check out my other great recipes.

Images are added very similarly to links, but the square brackets are preceded by ! and contain alt text for the visually impaired, and the () can contain a web address for an image.
